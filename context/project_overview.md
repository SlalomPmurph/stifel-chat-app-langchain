# Project Overview

## Stifel Financial Group Chat Application

### Purpose
This is a chat application designed for the Stifel Financial Group to enhance the capabilities of their financial advisors.

### Core Functionality
The application allows Stifel financial advisors to ask questions through a chat interface and receive answers related to their customers' financial information, accounts, and data.

### Technology Stack

#### Frontend
- **React**: Modern, component-based user interface
- **Chart.js**: Interactive data visualization and charting capabilities

#### Backend
- **Python**: Core backend language
- **LangChain**: AI/LLM integration framework for intelligent chat responses
- **Ollama**: Local LLM server for running models
- **Mistral**: Open-source LLM model for natural language processing

### Key Features

1. **Intelligent Chat Interface**
   - Financial advisors can ask natural language questions
   - Receive contextual answers related to customer information
   - AI-powered responses through LangChain integration

2. **Data Visualization**
   - Display charts and graphs directly in the chat interface
   - Chart.js integration for interactive visualizations
3. **Local AI Processing**
   - Ollama-powered local LLM for privacy and cost efficiency
   - Mistral model for high-quality financial analysis
   - No API costs or external data transmission
   - Customer data remains secure within infrastructure

4. **Customer-Centric Data**

3. **Customer-Centric Data**
   - Advisor-specific customer information
   - Real-time access to customer accounts and financial data
   - Secure, personalized data access per advisor

### Use Cases
- Advisors can query customer account information
- Generate visual representations of customer portfolios
- Analyze customer data through conversational AI
- Access insights through an intuitive chat interface
